🤖 JARVIS – AI Assistant

JARVIS is a smart voice-controlled AI assistant designed to automate daily tasks and improve user productivity. It listens to user commands and performs actions like playing music, opening applications, searching the web, and more in real-time.

🚀 Features
🎙️ Voice recognition and command processing
🌐 Web search and information retrieval
🎵 Play songs on YouTube
💻 Open applications automatically
🗣️ Text-to-speech responses
⚡ Fast and real-time interaction
🛠️ Technologies Used
🐍 Python
🧠 Speech Recognition
🔊 Pyttsx3 (Text-to-Speech)
🌐 PyWhatKit (YouTube & Web Automation)
🎨 EEL (Frontend + Backend Integration)
📂 Project Structure
JARVIS/
│── recognize.py       # Main controller (UI + Face Auth)
│── main.py            # Core assistant logic
│── www/               # Frontend files (HTML, CSS, JS)
│── requirements.txt   # Dependencies
⚙️ Installation
Clone the repository
git clone https://github.com/your-username/jarvis-ai.git
Navigate to project folder
cd jarvis-ai
Install dependencies
pip install -r requirements.txt
▶️ How to Run
python recognize.py
The UI will open automatically
Speak commands like:
“Play song on YouTube” 🎵
“Open Chrome” 🌐
“Search something” 🔍
🎯 Project Objective

The main goal of JARVIS is to create an intelligent, interactive, and user-friendly assistant that simplifies daily tasks using AI and automation.
